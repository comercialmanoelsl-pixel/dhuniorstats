# DhuniorStats V11 — Multi-Fontes

Inclui:
- API-Football principal;
- ESPN para lista complementar;
- base interna de contexto: clássicos, altitude e observações;
- jogadores que mais finalizam via API-Football;
- preparação para SportMonks via SPORTMONKS_KEY;
- preparação para notícias via NEWS_API_KEY;
- sem inventar tendência quando não houver base real.

Render:
Build Command: npm install
Start Command: npm start

Environment:
API_FOOTBALL_KEY = sua chave API-Football

Opcionais:
SPORTMONKS_KEY = chave SportMonks
NEWS_API_KEY = chave NewsAPI
