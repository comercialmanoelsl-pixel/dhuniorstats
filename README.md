# DhuniorStats V17 — Prováveis Vencedores

Novidades:
- Aba “Prováveis vencedores”
- Probabilidade contextual por time
- Motivos sem ícones de check
- Durante o jogo, previsões ficam verdes/vermelhas conforme batem ou não:
  - vitória parcial
  - pressão
  - finalizações
  - escanteios
- Estatísticas reais continuam separadas das tendências
- NewsAPI entra para detectar pressão/necessidade quando houver notícia

Variáveis:
API_FOOTBALL_KEY
NEWS_API_KEY
OPENWEATHER_KEY
THESPORTSDB_KEY
SPORTMONKS_KEY
FOOTBALL_DATA_KEY
ODDS_API_KEY

Build:
npm install

Start:
npm start
