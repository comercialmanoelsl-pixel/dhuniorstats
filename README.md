# Meu Futebol Live V3

- ESPN unofficial: lista de jogos, placar e status.
- API-Football/API-Sports: estatísticas ao clicar em "Ver estatísticas completas".

Render:

- Start Command: npm start
- Environment variable obrigatória:
  API_FOOTBALL_KEY = sua chave da API-Football
